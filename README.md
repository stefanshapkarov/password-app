
## Installation and Running in development

This is a Vue 3 + Vite project

First enter the project dir
```bash
cd project-location
```

Install necessary dependencies
```bash
npm install
```

Run the project
```bash
npm run dev
```

If by any chance zxcvbn is not installed, simply run
```bash
npm install zxcvbn
```

Then re-run the project
```bash
npm run dev
```
